
# 介護福祉士 毎日3問トレーナー

忙しい日でも “毎日3問” で合格力を積み上げる、介護福祉士向け学習アプリです。  
学生／社会人のスキマ時間に合わせて、1問1分ペースで知識を定着させます。

## 特徴
- 毎日3問で続けやすい
- 分野バランス／丁寧な解説
- 連続学習（Streak）で習慣化をサポート
